# :globe_with_meridians:概要

## 使用言語
- C/C++ (Arduino IDEによるコーディング)

## 開発背景
　私が長年行っている音楽活動において、ライブでのサウンドシステムの中核はMIDIによる一括制御となっている。制御対象となる機材の増加とライブ中に使用するサウンドの増加により、ここ最近は市販のMIDIスイッチングシステムでは事足りなくなってしまっていた。

　これを解消するため、自分の用途にフィットするMIDIスイッチングシステムを自分で制作すればいいと思い立ち、本ツールの作成に至った。
 
## 開発概要
　本ツールは、Arduino Mega2560に書き込む想定でArduino IDEを用いて制作した。また、マイコンのソースのみならず、スイッチャーの筐体作成およびMIDI送信回路や7セグメントディスプレイ、各種スイッチ等の回路もすべて自作したものとなる。

 また、本ツールは自分の用途に足るよう、以下の仕様を盛り込んだ。
 - 多機能であるがコンパクトさを維持するため、フットスイッチは横並びで4つとする。
 - 1つのスイッチにつき、押下ごとに表パッチ/裏パッチの選択ができるようにし、長押しすることでバイパスモードへと切り替わるようにする。
 - バンクスイッチは上スイッチ/下スイッチ式ではなく、1スイッチでの短押し/長押しで上下が切り替わるようにする。
 - 筐体には、本機単体でプリセットの書き換えができるよう、上下左右キーとメニューボタンを実装する。
 - 現在の選択状況や編集状況が分かるように7セグメントディスプレイを実装する。
 - PC側でcsv形式のパッチリストを作成し、それをI<sup>2</sup>C通信で受信して記録する機能を実装する。

<br/>

　本機のプリセットデータ構造は以下の通り
| バンク | パッチ | 表/裏 | Program Change | Control Change No | Control Change Val |
| :---: | :---: | :---: | :---: | :---: | :---: |
| バンク1 | パッチ1～4 | 表 | PC 1～128 | CC1 No 1～128 | CC Val 1～128 |
|        |            |    |           | CC2 No 1～128 | CC Val 1～128 |
|        |            |    |           | CC3 No 1～128 | CC Val 1～128 |
|        |            |    |           | CC4 No 1～128 | CC Val 1～128 |
|        |            | 裏 | PC 1～128 | CC1 No 1～128 | CC Val 1～128 |
|        |            |    |           | CC2 No 1～128 | CC Val 1～128 |
|        |            |    |           | CC3 No 1～128 | CC Val 1～128 |
|        |            |    |           | CC4 No 1～128 | CC Val 1～128 |

## 実績
　**開発から2年ほどが経つが、年30本ペースでのライブ活動および自宅でのレコーディングにて現在でも活躍している**。ライブ一本につき25分～35分程度、また自動車での移動やライブハウスへの機材搬入による振動・衝撃も多い環境での使用であるが、ハード・ソフトともにトラブルなく現在まで継続使用中。

## 確認済み動作環境
- Arduino Mega2560 (USB通信によるI<sup>2</sup>通信の仕様や入出力端子の本数の違いにより、Mega2560以外での動作は非推奨)

## インストール方法
1. 本リポジトリをZIP形式でダウンロード。任意の場所に解凍してください。
2. 解凍したフォルダ名を「MIDI_Programmable_Controller」へ変更してください。(Arduino IDEはメインのソースと同じフォルダ名を要求するため)
3. Arduino IDEを起動してマイコンへの書き込みを行ってください。

>[!TIP]
>書き込み時には、同梱のlibフォルダ内のライブラリを、Arduino IDE側でライブラリとして指定してください。

## 使用方法
- 4つのフットスイッチはそれぞれパッチ1～4に対応しており、以下のように切り替わる。
短押し：表パッチと裏パッチの切り替え
長押し：表パッチからでも裏パッチからでもバイパスへ切り替え

- バンクスイッチは短押しで上方向、長押しで下方向へ切り替わる。

- メニューボタンを押すと次のようにモードが切り替わる。
  1.プログラムモード：各バンク、各パッチの選択時に送信されるMIDIメッセージを編集。
  2.バイパスプログラムモード：バイパスに切り替えた際に送信されるMIDIメッセージを編集。
  3.有効バンク数設定モード：有効にするバンク数を設定。最大バンク数は10。
  4.画面輝度設定モード：7セグメントディスプレイの輝度を設定。
  5.I<sup>2</sup>C通信による、対応アプリからのパッチデータ受信モード：USB接続したPCよりパッチデータを受信。

## 技術的ハイライト
1. I<sup>2</sup>C通信によるプリセットデータの受信を組み込むことで、10バンク×4パッチ×表/裏Program Change×CC1～4という膨大なデータを筐体のボタンでちまちま手動で登録する必要がないようにした。  
  ⇒Recieve.h参照

<br/>

2. フットスイッチ4つという最小限のレイアウトに抑えながらも、スイッチひとつにつき表/裏2パッチ使えるようにすることで、1バンクにつき8個のサウンドを切り替えられるようにした。  
  ⇒PlayMode.h参照

<br/>

3. 「MIDIメッセージの内部データ構造を、プログラムモードにおける階層構造と一致させる」「現在選択中のモードを変数に持たせ、処理の遷移の際に参照することで、外部からも内部からモードの状態が一致するようにする」というステートマシン的な設計で実装した。
  ⇒MIDI_Programmable_Controller.inoからの各種モード遷移、MidiMessage.h参照
